# XD-Mall
基于VUE开发的电商项目
node + express + mysql 实现后台数据管理
# com-wxbdhcm 为vue-cli文件 
# clone the project
git clone https://github.com/wxbdhcm/XD-Mall

# enter the project directory
cd vue-element-admin

# install dependency
npm install

# develop
npm run serve

# express 运行 
npm start || nodemon start
